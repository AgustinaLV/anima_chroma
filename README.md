# Anima Chroma — Portfolio Web

> Diseño que se siente.

Portfolio profesional de diseño de interiores, construido con HTML, Tailwind CSS y tipografías de Google Fonts. Incluye página de inicio con secciones de proyectos, filosofía y CTA, más una página de contacto dedicada.

---

## 🌐 Vista general

| Página | Descripción |
|---|---|
| `portfolio.html` | Página principal: hero, proyectos destacados, filosofía y CTA |
| `contacto.html` | Página de contacto con formulario y datos de contacto |

---

## 🚀 Tecnologías

- **HTML5** semántico
- **Tailwind CSS** (CDN con plugins `forms` y `container-queries`)
- **Google Fonts**: Instrument Serif + Outfit
- **Material Symbols** (iconografía)
- **Dark mode** activado por defecto (`class="dark"`)

---

## ✨ Características

- Diseño **dark-first** con paleta personalizada (`#ec5b13` naranja primario, fondo `#221610`)
- **Efectos hover** sobre imágenes de proyectos: escala, rotación y overlay en gradiente
- Navegación fija con **backdrop-blur** y bordes sutiles
- Tipografía display con `Instrument Serif` y body con `Outfit`
- **Formulario de contacto** con validación y diseño premium
- Navegación consistente entre páginas (Proyectos → `#proyectos`, Filosofía → `#filosofia`)

## 🛠️ Cómo usar

1. Cloná (o descargá) este repositorio.
2. Abrí `portfolio.html` en tu navegador, o usá una extensión como **Live Server** en VS Code.
3. No requiere build ni dependencias de Node.js. Todo corre en el navegador directamente.

```bash
git clone https://github.com/tu-usuario/anima-chroma-portfolio.git
cd anima-chroma-portfolio
# Abrí portfolio.html en el navegador
```
